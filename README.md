# 快速上手

## 方法一、运行 exe

[Release]()页面发布了已经打包好的 exe 可执行程序，包括图形化版本和命令行版本。

用户只需要双击 exe 就可以使用啦！

## 方法二、运行 python 脚本

### 使用前安装需要的包

```
pip install -r requirements.txt -i https://pypi.org/simple/
```

### 使用图形界面运行:

```
python bilinovel_gui.py
```

### 使用命令行模式运行:

```
python bilinovel.py
```
